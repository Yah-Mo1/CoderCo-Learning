Binary, Octal and String Representation:

| Binary | Octal | String/Character |
| ------ | ----- | ---------------- |
| 000    | 0     | ---              |
| 001    | 1     | --x              |
| 010    | 2     | -w-              |
| 011    | 3     | -wx              |
| 100    | 4     | r--              |
| 101    | 5     | r-x              |
| 110    | 6     | rw-              |
| 111    | 7     | rwx              |

Common Binary-Octal Conversions:

- Binary 000 -> Octal 0
- Binary 001 -> Octal 1
- Binary 010 -> Octal 2
- Binary 011 -> Octal 3
- Binary 100 -> Octal 4
- Binary 101 -> Octal 5
- Binary 110 -> Octal 6
- Binary 111 -> Octal 7

Note:

- Each octal digit represents 3 binary digits
- Binary numbers are base-2 (0,1)
- Octal numbers are base-8 (0-7)
